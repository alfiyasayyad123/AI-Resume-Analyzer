# AI Resume Analyzer

## 📌 Project Description
AI Resume Analyzer is a Java-based application that helps users analyze their resumes. It calculates the ATS score, identifies skills, and provides suggestions to improve the resume.

## ✨ Features
- Upload Resume (PDF)
- Resume Analysis
- ATS Score
- Skills Detection
- AI Suggestions
- User Login & Registration
- Dashboard

## 🛠 Technologies Used
- Java
- HTML
- CSS
- JavaScript
- Spring Boot
- MySQL
- Git & GitHub

## 🚀 How to Run
1. Clone the repository.
2. Open the project in VS Code or IntelliJ.
3. Configure the MySQL database.
4. Run the project.
5. Open the application in your browser.
